# GenAI Fundamentals — Session 3

A 1-hour visual introduction to Generative AI concepts for a mixed technical audience.

## About This Session

This is **Session 3** in the Python-AI series:
- Sessions 1-2: Python basics
- **Session 3: GenAI fundamentals (this session)**
- Future sessions: Building GenAI and Agentic AI workflows in Python

## What You'll Learn

1. How Large Language Models (LLMs) actually work
2. Why LLMs have limitations (hallucinations, knowledge cutoff)
3. How Retrieval-Augmented Generation (RAG) solves these problems
4. The role of embeddings, vectors, and chunking
5. Prompt engineering essentials
